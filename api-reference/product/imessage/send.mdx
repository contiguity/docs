<Note>
    Only customers who lease numbers that support iMessage can use this endpoint.
</Note>

<Warning>
Phone numbers must be in E.164 format with country codes.
</Warning>

<Info>
Technically, iCloud emails are supported... but their behavior is unpredictable.
</Info>

## Message formatting

Format message text with markdown-style syntax. Supported styles:

| Style | Syntax |
| --- | --- |
| **Bold** | `**text**` |
| _Italics_ | `_text_` or `*text*` |
| ~~Strikethrough~~ | `~~text~~` |
| __Underline__ | `__text__` |
| <strong><u><s><em>Combined</em></s></u></strong> | `**__~~_text_~~__**`|
